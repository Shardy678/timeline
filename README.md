# Установка
Шаги
1. Склонируйте репозиторий:

```bash
git clone https://github.com/Shardy678/timeline.git
```

2. Перейдите в директорию проекта:

```bash
cd timeline
```
Установите зависимости:

```bash
npm install
```

3. Соберите проект

```bash
npm run build
```

4. Откройте index.html, который находится в папке dist/
